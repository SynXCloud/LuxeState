# LuxeState — Premium Real Estate Platform 🏛️

**LuxeState** is a meticulously designed, ultra-premium real estate platform tailored for discerning buyers looking for luxury residences, penthouses, and modern villas across top tier cities in India. 

The project aims to deliver a high-end, seamless user experience combining aesthetic excellence with powerful functional tools to aid in property discovery and financial planning.

---

## 🌟 Key Features

* **Premium Aesthetic UI/UX**: Cinematic visuals, sleek dark-mode design, smooth scroll animations, and glassmorphism elements create a luxurious feel.
* **AI-Powered "Smart Match"**: A quick, intuitive 4-step wizard that helps users instantly find properties tailored to their specific lifestyle, budget, and configuration preferences.
* **Curated Property Catalog**: Detailed property cards with high-resolution imagery, vital specifications (Beds, Baths, Area, Parking), and RERA verification tags.
* **Advanced Filtering System**: Real-time filtering by Location, Configuration (BHK), Price Range, Amenities, and Construction Status.
* **Property Comparison Tool**: An interactive tool allowing users to select up to 3 properties and compare them side-by-side on all major metrics.
* **Dynamic EMI Calculator**: A built-in financial tool featuring interactive sliders and a real-time updating visual breakdown (pie chart) of Principal vs. Interest.
* **Wishlist functionality**: Users can easily save their favorite properties with a single click.
* **Interactive AI Assistant (Aria)**: A responsive chatbot ready to guide users on locations, budgets, EMI estimates, and scheduling callbacks.

## 🛠️ Technology Stack

* **HTML5**: Semantic structure and accessibility.
* **CSS3 (Vanilla)**: Custom CSS variables for theme management, Flexbox/Grid layouts, micro-animations, and responsive design. No heavy frameworks to ensure maximum performance and pixel-perfect control.
* **JavaScript (Vanilla)**: DOM manipulation, filtering logic, financial calculations, state management, and interactive components.

## 🚀 Getting Started

To run the project locally, you simply need to serve the static files:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/SynXCloud/LuxeState.git
   cd LuxeState
   ```

2. **Run a local server**:
   You can use Python's built-in HTTP server or any live server extension in your IDE.
   ```bash
   python3 -m http.server 8080
   ```

3. **View the site**:
   Open your browser and navigate to `http://localhost:8080`.
